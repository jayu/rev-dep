# Quick Start

Use this section when you want the first useful result from `rev-dep` without learning the full config surface first.

## Choose Your Starting Point

- Want repeatable checks in CI: install the tool and run `rev-dep config init`
- Want to inspect the project immediately: use one of the exploratory commands like `entry-points`, `circular`, or `node-modules unused`

## Recommended First Flow

1. Install `rev-dep` locally in the repo.
2. Generate a starter config with `rev-dep config init`.
3. Define production and development entry points.
4. Run `rev-dep config run`.
5. Add more checks once the first results make sense.
