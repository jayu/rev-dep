# Trace direct importers

Use:

```bash
rev-dep imported-by --file src/utils/math.ts
```

This shows direct importers only. Use it when you want fast local impact analysis without tracing the full path from an entry point.

## Useful flags

```bash
rev-dep imported-by --file src/utils/math.ts --count
rev-dep imported-by --file src/utils/math.ts --list-imports
```

`--list-imports` is especially useful when the same file is imported under different request strings and you need to see the raw requests.
