Count actual lines of code in the project excluding comments and blank lines

```
rev-dep lines-of-code [flags]
```

### Examples

```
rev-dep lines-of-code
```

### Options

```
  -c, --cwd string   Directory to analyze (default "$PWD")
  -h, --help         help for lines-of-code
```
