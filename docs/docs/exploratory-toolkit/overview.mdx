# Exploratory Toolkit

Use the exploratory commands when you want to answer one dependency question quickly without building or changing a config.

Typical questions:

- What are the project entry points?
- Which files does this entry point pull in?
- Who imports this file?
- Is there a path from one file to another?
- Which packages are used or missing?

These commands are ideal for refactoring, debugging, and understanding unfamiliar code.
